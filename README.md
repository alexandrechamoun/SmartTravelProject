Project Overview:
SmartTravel is a console-based travel agency management system. This assignment extends A1 by adding:
Custom exception handling (6 exception classes)
Input validation on all domain objects
File I/O persistence via CSV files
Search/filter functionality
An HTML dashboard with charts

How to Run:

Navigate to src/driver/Driver.java
Right-click → Run
The console will display a welcome message and prompt:
Option 1 — Interactive menu (manage clients, trips, transport, accommodations)

Option 2 — Predefined scenario (demonstrates all features and exceptions)
